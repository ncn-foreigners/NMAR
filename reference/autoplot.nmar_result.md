# Default ggplot2 autoplot for NMAR results

Quick ggplot2 visualizations for result objects.

## Usage

``` r
# S3 method for class 'nmar_result'
autoplot(object, type = c("weights", "fitted", "constraints"), ...)
```

## Arguments

- object:

  An object of class \`nmar_result\` or a subclass.

- type:

  One of "weights", "fitted", or "constraints".

- ...:

  Ignored.

## Value

A \`ggplot\` object.
