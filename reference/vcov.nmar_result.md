# Variance-covariance for NMAR results

Variance-covariance for NMAR results

## Usage

``` r
# S3 method for class 'nmar_result'
vcov(object, ...)
```

## Arguments

- object:

  An object of class \`nmar_result\`.

- ...:

  Ignored.

## Value

A 1x1 numeric matrix (the variance of the primary estimate).
