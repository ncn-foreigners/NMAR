# Wald confidence interval for base NMAR results

Wald confidence interval for base NMAR results

## Usage

``` r
# S3 method for class 'nmar_result'
confint(object, parm, level = 0.95, ...)
```

## Arguments

- object:

  An object of class \`nmar_result\`.

- parm:

  Ignored.

- level:

  Confidence level.

- ...:
